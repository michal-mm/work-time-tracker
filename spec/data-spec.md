Each day (example 2026-02-23) represents any type of day.
THe properties in the file will represent whether it is
a regular work day (usually Mon-Fri), weekend (Sat & Sun),
public holiday (a day, but wth 0 hours expected to work)
or vacation (days off) or even sick leaves.

File structure will be JSON (the most efficient way to keep 
some structure), name will be the date format yyyy-mm-dd.json,
example 2026-02-26.json
